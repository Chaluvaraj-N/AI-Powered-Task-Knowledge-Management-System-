# Frontend — AI-Powered Task & Knowledge Management System

React (Vite) + Tailwind CSS frontend. See the [root README](../README.md) for
full setup instructions, tech stack, and API documentation.

Quick start:

```bash
npm install
npm run dev
```

Runs at http://localhost:5173 and proxies `/api/*` to the FastAPI backend at
http://localhost:8000 (see `vite.config.js`).
